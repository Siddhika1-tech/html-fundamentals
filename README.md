# HTML Fundamentals

A beginner-friendly HTML project demonstrating the fundamental concepts and core elements of HTML5.

## Features

- HTML document structure
- Headings (`<h1>` to `<h6>`)
- Paragraphs
- Text formatting
- Line breaks and horizontal rules
- Quotations
- Superscript and subscript
- Abbreviations
- Images
- Basic HTML elements

## Technologies Used

- HTML5

## Project Structure

```
html-fundamentals/
├── index.html
├── README.md
└── LICENSE
```

## Getting Started

1. Clone this repository:
   ```bash
   git clone https://github.com/Siddhika1-tech/html-fundamentals.git
   ```

2. Open `index.html` in any modern web browser.

## Learning Outcomes

This project helps beginners understand:

- HTML document structure
- Common HTML tags and elements
- Text formatting
- Organizing web page content
- Building a simple static webpage
